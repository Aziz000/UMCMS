#Universal Medical Centers Management System (UMCMS)

> SE3087 Software Engineering Project — Prince Sattam bin Abdulaziz University

## Overview
UMCMS is a web-based healthcare management system designed to digitize and integrate 
the core workflows of a universal medical center — from patient registration to 
clinical consultation, pharmacy dispensing, billing, and administrative analytics.

## Key Features
- Patient registration & EMR management
- Real-time appointment booking (no double-booking)
- Digital prescriptions with drug interaction alerts
- Automated invoice generation
- Admin analytics dashboard (multi-branch support)
- Role-based access control (5 roles)

## System Users
| Role | Responsibility |
|------|---------------|
| Patient | Register, book appointments, view records |
| Receptionist | Check-ins, billing assistance |
| Doctor | Consultations, EMR, prescriptions |
| Pharmacist | Dispense medications |
| Administrator | System config, analytics, branch management |

## Architecture
Three-tier: Presentation Layer → Business Logic (REST API) → Data Layer (Relational DB)
